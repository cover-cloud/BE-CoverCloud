API 테스트

📝 사용법

  1. OAuth 로그인: 카카오/네이버 버튼 클릭하여 실제 로그인
  2. 테스트 토큰: 개발용 토큰 즉시 발급 (새 탭에서 열림)
  3. API 테스트: 발급받은 토큰을 입력하고 API 테스트

🔗 직접 API 호출 (curl)


# 테스트 토큰 발급
curl http://localhost:8080/api/test/token?userId=1

# 내 커버 목록
curl -H "Authorization: Bearer YOUR_TOKEN" http://localhost:8080/api/cover/my

# 좋아요
curl -X POST -H "Authorization: Bearer YOUR_TOKEN" http://localhost:8080/api/cover/1/like

# 좋아요 취소  
curl -X POST -H "Authorization: Bearer YOUR_TOKEN" http://localhost:8080/api/cover/1/unlike
